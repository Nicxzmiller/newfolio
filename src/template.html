Covid19 Live Statistics

This website shows the various Covid-19 virus statistics based on different countries. The results are based on real time staticstics depending on the time each country logs in their data. This project was built with ReactJs framework, React-Bootstrap, HTML5, and CSS. You have the ability to change the view from light mode to dark mode and also search functionality takes effect immediately you type the first letter of a country. you can visit the site for live experience.
